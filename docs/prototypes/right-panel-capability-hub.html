Capability Hub Draft

Right panel as a live capability hub

This draft stops treating Skills as a standalone page. The right panel becomes a session-level capability hub: clickable blocks first, richer detail second, and a recent-resources convergence area that catches images, HTML prototypes, notes, and touched files from the current AI flow.

Layout Recipe

Same capability-hub idea, three different first-screen emphases. Every button here is live and updates the phone preview immediately.

Detail Interaction

Clicking a capability block can open richer content in three real ways: overlay, inline expand, or replace-the-panel.

Inject Demo Resources

Simulate the hub receiving new AI results. This shows whether the recent-resources convergence area still feels useful once the panel becomes busy.

Interaction Notes

Capability blocks are the first layer The first thing you see is not a list page. It is a set of useful blocks like Skills History, Generated Images, Artifacts, and Touched Files.
Recent resources are the convergence layer Images, files, HTML prototypes, and notes all end up in one place so the user can quickly revisit what the AI just produced.
Settings stays separate Installed Skills remains in Settings. This hub is about current-session capability and output, not machine inventory.
